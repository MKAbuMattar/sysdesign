# Assets

| File | Use |
| --- | --- |
| `logo.svg` | Primary lockup (mark + wordmark). README header, docs. |
| `icon.svg` | Square tile, 280×280. Marketplace listing, social preview, app icon. |
| `favicon.svg` | Simplified mono mark, transparent. Favicon, npm avatar. |
| `favicon-tile.svg` | White mark on dark tile. Favicon on light backgrounds. |
| `commands/*.svg` | Per-command glyphs (100×100), one hue each: explain, compare, review, choose, estimate, tradeoffs, diagram, interview, cheatsheet, help. |

Wordmark uses a system font stack so it renders on any machine. Accent palette: blue `#5B8DEF` · teal `#3BB89A` · amber `#E0A458` · ink `#2D3748`.
